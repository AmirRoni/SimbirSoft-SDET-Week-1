# UI автотесты для automationteststore.com

Проект выполнен в рамках задания «Практикум SDET: задание UI».

Объект тестирования:
https://automationteststore.com/

## Описание

В проекте планируется реализовать UI-автотесты для интернет-магазина Automation Test Store по предоставленному чек-листу.

Тесты будут покрывать основные пользовательские сценарии:
- проверка сортировки товаров в категории
- проверка поисковой выдачи и работы корзины
- проверка добавления и удаления товаров в корзине

Также в проект будут добавлены тест-кейсы в формате Markdown.

## Технологии

- Python 3.10
- pytest
- Selenium WebDriver (selenium==4.43.0)
- Chrome (Версия: 147.0.7727.57 (официальная сборка) (64 бит))

Дополнительно планируется:
- Allure Report
- параллельный запуск тестов
- запуск в CI/CD

## Покрытие по чек-листу

Планируется реализовать следующие проверки:

### 1. Фильтрация в категориях
В одной из категорий, содержащей не менее 4 товаров:
- проверить сортировку товаров по имени:
  - по возрастанию
  - по убыванию
- проверить сортировку товаров по стоимости:
  - по возрастанию
  - по убыванию

### 2. Проверка поисковой выдачи и корзины
- выполнить поиск по запросу `shirt`
- отсортировать результаты по наименованию
- добавить в корзину 2-й и 3-й товар из выдачи
- использовать случайное количество товаров
- перейти в корзину
- найти самый дешевый товар
- удвоить его количество
- проверить итоговую сумму

### 3. Проверка корзины
- добавить в корзину 5 случайных товаров с главной страницы
- использовать случайное количество для каждого товара
- перейти в корзину
- удалить все товары с чётными номерами в таблице корзины
- проверить итоговую стоимость

## Запуск проекта

Команды будут уточнены после реализации проекта.

Предварительно:

Установка зависимостей:
`pip install -r requirements.txt`

Запуск всех тестов:
`pytest -v`

## Отчётность

После реализации проекта планируется добавить поддержку Allure Report.

Предварительные команды:

Запуск тестов с генерацией результатов Allure:
`pytest --alluredir=allure-results`

Открытие отчёта:
`allure serve allure-results`
