# Тест-кейсы для automationteststore.com

## Общая информация

**Объект тестирования:** https://automationteststore.com/  
**Тип тестирования:** UI  
**Инструмент автоматизации:** Selenium WebDriver + pytest

---

## ТК-1. Проверка сортировки товаров в категории по имени и цене

**Проверяемое требование:**  
Фильтрация в категориях: проверить сортировку товаров по имени и стоимости в обоих направлениях — по возрастанию и убыванию.

**Предусловия:**
1. Открыт сайт automationteststore.com.
2. Выбрана категория, в которой отображается не менее 4 товаров.
3. Пользователь находится на странице категории.

**Шаги:**
1. Открыть страницу выбранной категории.
2. Убедиться, что в категории отображается не менее 4 товаров.
3. Выбрать сортировку по имени по возрастанию.
4. Считать названия товаров в порядке отображения.
5. Проверить, что названия отсортированы в алфавитном порядке по возрастанию.
6. Выбрать сортировку по имени по убыванию.
7. Считать названия товаров в порядке отображения.
8. Проверить, что названия отсортированы в алфавитном порядке по убыванию.
9. Выбрать сортировку по цене по возрастанию.
10. Считать цены товаров в порядке отображения.
11. Проверить, что цены отсортированы по возрастанию.
12. Выбрать сортировку по цене по убыванию.
13. Считать цены товаров в порядке отображения.
14. Проверить, что цены отсортированы по убыванию.

**Ожидаемый результат:**
1. На странице категории отображается не менее 4 товаров.
2. После выбора сортировки по имени по возрастанию товары отображаются в корректном алфавитном порядке.
3. После выбора сортировки по имени по убыванию товары отображаются в обратном алфавитном порядке.
4. После выбора сортировки по цене по возрастанию товары отображаются от меньшей цены к большей.
5. После выбора сортировки по цене по убыванию товары отображаются от большей цены к меньшей.

## ТК-2. Проверка поисковой выдачи и пересчёта итоговой стоимости в корзине

**Проверяемое требование:**  
В поиск ввести "shirt", отсортировать по наименованию и положить в корзину 2 и 3 товар из выдачи с рандомным количеством. В корзине найти самый дешевый товар, удвоить его количество, проверить итоговую сумму.

**Предусловия:**
1. Открыт сайт automationteststore.com.
2. Пользователь находится на главной странице сайта.
3. По запросу `shirt` отображается не менее 3 товаров.

**Шаги:**
1. Открыть главную страницу сайта.
2. Ввести в поиск значение `shirt`.
3. Запустить поиск.
4. Убедиться, что поисковая выдача содержит не менее 3 товаров.
5. Выбрать сортировку результатов по имени по возрастанию.
6. Открыть 2-й товар из поисковой выдачи.
7. При необходимости выбрать обязательные параметры товара.
8. Установить случайное количество товара.
9. Добавить товар в корзину.
10. Повторно выполнить поиск по запросу `shirt`.
11. Снова выбрать сортировку по имени по возрастанию.
12. Открыть 3-й товар из поисковой выдачи.
13. При необходимости выбрать обязательные параметры товара.
14. Установить случайное количество товара.
15. Добавить товар в корзину.
16. Перейти в корзину.
17. Считать список товаров в корзине, их цены и количества.
18. Найти товар с минимальной ценой за единицу.
19. Увеличить его количество в 2 раза относительно текущего.
20. Обновить корзину.
21. Повторно считать товары, их цены и количества.
22. Считать стоимость доставки, отображаемую в корзине.
23. Рассчитать ожидаемую итоговую стоимость как сумму стоимости всех товаров и стоимости доставки.
24. Считать фактическую итоговую стоимость, отображаемую в корзине.
25. Сравнить рассчитанную и отображаемую итоговые стоимости.

**Ожидаемый результат:**
1. Поиск по запросу `shirt` выполняется успешно.
2. В выдаче отображается не менее 3 товаров.
3. 2-й и 3-й товары после сортировки по имени успешно добавляются в корзину.
4. Для товаров с обязательными опциями они корректно выбираются перед добавлением.
5. В корзине корректно определяется самый дешевый товар.
6. После увеличения количества этого товара в 2 раза корзина успешно обновляется.
7. Итоговая стоимость корзины с учётом доставки совпадает с рассчитанной на основе цен, количеств товаров и стоимости доставки.

## ТК-3. Проверка удаления чётных товаров из корзины и пересчёта итоговой стоимости

**Проверяемое требование:**  
Добавить в корзину 5 рандомных товаров с главной страницы с рандомным количеством. В корзине удалить все чётные по порядку товары из корзины, проверить итоговую стоимость.

**Предусловия:**
1. Открыт сайт automationteststore.com.
2. Пользователь находится на главной странице сайта.
3. На главной странице доступно не менее 5 товаров.

**Шаги:**
1. Открыть главную страницу сайта.
2. Считать список товаров, отображаемых на главной странице.
3. Убедиться, что на странице доступно не менее 5 товаров.
4. Случайным образом выбрать 5 различных товаров.
5. Для каждого из выбранных товаров:
   1. Открыть карточку товара.
   2. При необходимости выбрать обязательные параметры товара.
   3. Установить случайное количество.
   4. Добавить товар в корзину.
6. Перейти в корзину.
7. Убедиться, что в корзине отображается 5 товаров.
8. Определить товары, стоящие на чётных позициях в таблице корзины: 2-й, 4-й и далее.
9. Удалить все товары, стоящие на чётных позициях.
10. Дождаться обновления корзины после каждого удаления.
11. Считать оставшиеся в корзине товары, их цены и количества.
12. Считать стоимость доставки, отображаемую в корзине.
13. Рассчитать ожидаемую итоговую стоимость как сумму стоимости оставшихся товаров и стоимости доставки.
14. Считать фактическую итоговую стоимость, отображаемую в корзине.
15. Сравнить рассчитанную и отображаемую итоговые стоимости.

**Ожидаемый результат:**
1. С главной страницы успешно добавляются 5 случайных товаров.
2. Для товаров с обязательными опциями они корректно выбираются перед добавлением в корзину.
3. В корзине отображается 5 товаров до удаления.
4. Все товары, стоящие на чётных позициях, успешно удаляются.
5. После удаления в корзине остаются только товары, стоящие на нечётных позициях.
6. Итоговая стоимость корзины с учётом доставки после удаления совпадает с рассчитанной на основе оставшихся товаров и стоимости доставки.
